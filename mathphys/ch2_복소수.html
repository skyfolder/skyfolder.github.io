복소수와 대기과학

이 글은 Boas의 Chapter 2 (Complex Numbers) 내용을 바탕으로, 대기과학과 학부 2학년 수준에 맞추어 재구성한 수업 교재 초안입니다.[2] 전체 흐름은 원문 구조(정의 → 복소평면 → 극형식 → 복소대수 → 응용)를 따르되, 약 30% 분량을 대기과학 관련 예제와 해설로 채웠습니다.[2]

1. 복소수의 도입과 대기과학에서의 필요성

1.1 복소수의 정의

2차방정식 \(az^2 + bz + c = 0\)의 해는 판별식 \(d < 0\)일 때 실수 범위에서 정의되지 않습니다. 이를 위해 새로운 수 \(i = \sqrt{-1}\)를 도입합니다.[2]

  • \(i^2 = -1\)
  • \(z = x + iy\) 형태의 수를 복소수(Complex Number)라 부릅니다.
  • \(x\): 실수부 (\(\mathrm{Re}\,z\)), \(y\): 허수부 (\(\mathrm{Im}\,z\))

1.2 대기과학에서 복소수가 왜 필요한가

대기과학에서 복소수는 주로 파동, 진동, 선형 시스템을 다룰 때 필수적입니다.

  1. 선형 파동 해석: Rossby 파나 중력파의 해를 \(e^{i(kx-\omega t)}\) 꼴로 표현하면 미분이 단순한 곱셈으로 바뀌어 계산이 간편해집니다.[2]
  2. 복소 증폭률·감쇠율: 점성이나 마찰이 포함되면 진동수 \(\omega\)가 복소수가 되며, 허수부는 파동의 성장이나 감쇠를 나타냅니다.[2]
  3. 관측 시스템: 레이더나 라이다의 신호 처리에서 위상과 진폭을 동시에 다루기 위해 사용됩니다.

2. 복소평면과 극좌표 표현

2.1 복소평면 (Complex Plane)

[Image of complex plane Argand diagram]

복소수 \(z = x + iy\)는 2차원 평면상의 점 \((x,y)\)로 나타낼 수 있습니다. 이 평면을 복소평면(Argand Plane)이라 부르며, \(x\)-축을 실수축, \(y\)-축을 허수축이라 합니다.[2]

2.2 직각좌표형과 극형식

점 \((x,y)\)를 극좌표 \(r,\theta\)로 변환하면 \(x = r\cos\theta, y = r\sin\theta\)입니다. 오일러 공식을 이용하면 복소수를 다음과 같이 극형식으로 쓸 수 있습니다.[2]

\[ z = r(\cos\theta + i\sin\theta) = re^{i\theta} \]
  • \(r = |z|\): 복소수의 크기 (Modulus)
  • \(\theta\): 복소수의 위상 (Argument)
예제 1: 극형식으로 나타내기

\(z = -1 - i\)를 극형식으로 표현해 보자.

  1. 크기: \(r = \sqrt{(-1)^2 + (-1)^2} = \sqrt{2}\)
  2. 위상: 점이 3사분면에 있으므로 \(\theta = 5\pi/4\)
  3. 결과: \(z = \sqrt{2} e^{i \frac{5\pi}{4}}\)

2.3 대기 파동의 위상과 복소평면

예제 2: 단일 주파수 파동의 복소 표현

중위도 선형 중력파 \(\eta(x,t) = A \cos(kx - \omega t)\)는 복소 표현으로 다음과 같습니다.

\[ \eta(x,t) = \mathrm{Re}\{ A e^{i(kx-\omega t)} \} \]

복소평면에서 이는 반지름이 \(A\)이고 각도가 \(kx-\omega t\)인 점이 원 위를 회전하는 것으로 해석할 수 있습니다. 이는 파동 방정식을 다루는 강력한 도구가 됩니다.[2]

3. 복소수의 대수: 사칙연산과 공액

3.1~3.3 연산 규칙

두 복소수 \(z_1, z_2\)에 대하여:

  • 곱셈 (극형식): 크기는 곱하고, 위상은 더한다. (\(r_1 r_2 e^{i(\theta_1 + \theta_2)}\))
  • 나눗셈 (극형식): 크기는 나누고, 위상은 뺀다. (\((r_1/r_2) e^{i(\theta_1 - \theta_2)}\))
  • 공액복소수: \(\bar z = x - iy\) (허수부의 부호 반대)
예제 5: 곱셈–나눗셈과 위상 변화

\(\frac{(1+i)^2}{1-i}\) 를 계산해 보자.

극형식으로 변환하면 \(1+i = \sqrt{2}e^{i\pi/4}\), \(1-i = \sqrt{2}e^{-i\pi/4}\) 이므로:

\[ \frac{(\sqrt{2}e^{i\pi/4})^2}{\sqrt{2}e^{-i\pi/4}} = \frac{2e^{i\pi/2}}{\sqrt{2}e^{-i\pi/4}} = \sqrt{2}e^{i(\frac{\pi}{2} + \frac{\pi}{4})} = \sqrt{2}e^{i\frac{3\pi}{4}} \]

이는 직각좌표계로 \(-1+i\) 에 해당합니다.

4. 복소지수와 오일러 공식, 대기 파동

4.1~4.3 오일러 공식과 파동 방정식

오일러 공식 (Euler's Formula): \[ e^{i\theta} = \cos\theta + i\sin\theta \]
예제 6: 선형 파동의 복소 관점 (미분을 곱셈으로)

파동해를 \(\eta = \hat\eta e^{i(kx-\omega t)}\)라 가정할 때, 미분 연산은 대수적 곱셈으로 변환됩니다.

  • 공간 미분 \(\partial_x \rightarrow ik\)
  • 시간 미분 \(\partial_t \rightarrow -i\omega\)

예를 들어, 파동 방정식 \(\frac{\partial^2 \eta}{\partial t^2} = c^2 \frac{\partial^2 \eta}{\partial x^2}\)에 대입하면, 미분방정식을 풀지 않고도 \((-i\omega)^2 = c^2 (ik)^2\) 라는 분산 관계식(\(\omega^2 = c^2 k^2\))을 즉시 얻을 수 있습니다.

5. 복소 멱급수와 수렴 영역

5.1 복소 멱급수와 디스크 수렴

복소 멱급수 \(\sum a_n z^n\)는 복소평면 상의 원판(Disk) 영역 \(|z| < R\) 내부에서 수렴합니다.[2]

5.2 대기과학 응용: β-평면 근사

코리올리 파라미터 \(f(\phi) = 2\Omega\sin\phi\)를 기준 위도 \(\phi_0\) 근처에서 전개하는 것은 복소 테일러 급수의 응용입니다. \[ f(\phi_0 + \delta) \approx f_0 + \beta y \quad (\text{여기서 } \beta = \partial f / \partial y) \] 작은 매개변수 \(\delta\)에 대한 급수 전개는 대기역학의 핵심 도구입니다.

6. 복소함수와 7. 로그/거듭제곱

복소 삼각함수와 쌍곡함수

\[ \sin z = \frac{e^{iz} - e^{-iz}}{2i},\quad \cos z = \frac{e^{iz} + e^{-iz}}{2} \] 대기과학에서는 수직적으로 감쇠하면서 수평으로 전파되는 파동(\(e^{ikx-\alpha z}\))을 다룰 때, 이를 복소 지수함수 혹은 쌍곡함수의 조합으로 해석합니다.

예제 9: 복소 성장률 (Stability Analysis)

대기 선형모드의 시간 진화가 \(z(t) = z_0 e^{\lambda t}\)이고, \(\lambda = \sigma + i\omega\) (복소수)일 때:

\[ e^{\lambda t} = e^{\sigma t} \cdot e^{i\omega t} \]
  • \(e^{\sigma t}\): 진폭의 성장(\(\sigma > 0\)) 또는 감쇠(\(\sigma < 0\))
  • \(e^{i\omega t}\): 파동의 진동 및 위상 변화

8. 물리적 응용: 궤도와 임피던스

8.1 입자 궤도 (Particle Trajectory)

수면 파동 입자의 궤적 \(z(t) = a e^{i\omega t}\)는 복소평면에서 원운동을 나타냅니다. 속도 벡터는 \(\frac{dz}{dt} = i\omega z\)로, 위치 벡터와 90도 위상 차이가 있음을 알 수 있습니다.

8.3 대기 시스템의 "임피던스"

예제 10: 행성경계층(PBL)의 응답

경계층 내 바람의 변화를 \(\frac{dU}{dt} = -\alpha U + \text{Forcing}\) 형태로 근사할 때, 주파수 응답 분석 시 \(-i\omega + \alpha\) 형태의 분모가 등장합니다. 이는 전기회로의 복소 임피던스(저항+리액턴스)와 수학적으로 동일한 구조를 가지며, 대기가 외부 강제력에 대해 어떻게 반응하고 지연되는지를 설명해 줍니다.[2]
